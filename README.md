# My Walking Weather

## PURPOSE
Nowdays travelers around the world plan their trips ahead of time.
For that reason we have created an app that allows travelers to see the weather outlook for multiple cities in real time.

## BUILT WITH
* HTML
* CSS
* JAVASCRIPT
* OPENWEATHER API
